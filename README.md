### Running Locally

#### Server

To test while developing, you can run your code changes with:

```
./local.mjs connect
./local.mjs clientOpen
./local.mjs disconnect
./local.mjs CORS
./local.mjs message
```

#### Client

```
npm install -g local-web-server

ws --https
```
